# WalletTheme

RNBrainTree

## **Version 1.0 - Friday 19 June, 2021**

## Requirements

- Paypal Sandbox business account
- Fetch nonce of PayPal

## Installation:

Install the dependencies

```sh
1- cd WalletTheme
2- yarn && cd android && ./gradlew installRelease
3- react-native start
```
